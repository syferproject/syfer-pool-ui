# Syfer Pool UI

> UI for [Syfer mining pool](https://pool.syfer.network/)

## Requirements

- Node.js 16
- Configured AWS profile (for deployment only)

## Local Development

```shell
$ git clone https://github.com/bomb-on/syfer-pool-ui.git
$ cd syfer-pool-ui
$ npm install
$ npm start
```

## Deploy

```shell
$ npm run aws:deploy
```
